== Description ==
This plugin provides to show awesome team showcase to you post or pages just using shortcode.

use shortcode to show all members in page or post.
`
[team-members]
` 

To Show members by category, use shortcode.

`
[team-members category="category name"]
`
Where 'category name' is category of member group which to be list.


<strong>Features</strong>

* You can add/edit member detail same way as post add/edit.
* Responsive layout.
* Shortcode enabled.
* Easy to customize
* Easy to use.

== Installation ==
1. Upload the folder "custom-team-manager" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to WP-admin-> Awesome Team Member ->Add New Member , here we can add new member.
4. use shortcode to show all members in page or post

`
[team-members]
` 

To Show members by category , use shortcode.

`
[team-members category="category name"]
`
Where 'category name' is category of member group which to be list.
